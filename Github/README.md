# BitQuest - Explorador de Matrices

Videojuego arcade en consola desarrollado de forma híbrida utilizando **C** para el control lógico del estado del juego y **NASM (x86-64 Assembly)** para el procesamiento gráfico bidimensional.

## Requisitos del Sistema
* Windows 10/11 de 64 bits.
* Compilador GCC de 64 bits instalado en el PATH.
* Ensamblador NASM instalado en el PATH.

## Instrucciones de Compilación y Ejecución

1. Abra una terminal de comandos (`cmd`) en esta carpeta.
2. Ejecute el script automatizado de compilación:
   ```bash
   build.bat
